# CMS Frontend

This folder contains the static frontend for the Classroom Record System.

## How to run

From `cms-frontend`:

```powershell
npm run start
```

Then open the browser at:

```
http://localhost:8000
```

## Notes

- `cms-frontend` is not a Laravel application.
- There is no `artisan` file here.
- Use `php artisan` only inside `cms-backend`.
